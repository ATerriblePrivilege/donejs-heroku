# my-app

## Prereqs
* Node is installed
* NPM is installed
* Heroku Account is created

## Steps
1. Clone Me
2. `cd donejs-heroku`
3. `heroku login`
4. `heroku create`
5. `npm install`
6. `donejs build`
7. `git add --all`
8. `git commit -am 'Deploy it'`
9. `git push heroku master`
10. `heroku open`
